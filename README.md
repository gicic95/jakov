# E-commerce template
Овај шаблон сајт је рађен у **Next.js**-у

Помоћне библиотеке: 
### Tailwind
> Коришћен као замена за CSS.
### shadcn/ui
> Lightweight компонентска библиотека.
### Zustand
> State management.
### Zod
> Type management.
## Инсталација

За production, зову се две команде:
```
 npm run build
```
затим
```
npm run start
```
